- html 은 정적인 문서
- 요청이 들어오면 응답할 내용(html)을 만들어내는 프로그램이 필요
- 왜냐하면 유저가 원하는 것들이 다 다르니깐
- python코드를 실행하기위한 web framework : django
- java: spring
